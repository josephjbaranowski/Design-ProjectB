Buttons & Typography

Buttons are meant to stand out, help your users see them!

Default

By default all buttons will be the $brand-cta variable color chosen in the _custvariables.scss file. Secondary button color by default is $brandy-primary variable. Links can be turned to buttons by the btn class and buttons can be changed to links by the link class.

One size does not fill all scenarios, which is why you can change sizes btn-sm, btn-lg, btn-xl.
